أثبت جورج غامو أن أقل عدد من النيوكليوتيدات اللازمة لتشفير حمض أميني واحد هو ثلاثة نيوكليوتيدات. تسمى هذه الوحدات الثلاثية كودونات.

قام نيرنبيرغ ومساعدوه بتحديد بعض الحموض الأمينية التي تشفرها الكودونات عمليًا.

جدول الشيفرة الوراثية:

| الكودون | الحمض الأميني |
|---|---|
| AUG | ميثيونين (بدء) |
| UAA | توقف |
| UAG | توقف |
| UGA | توقف |
| Phe | فينيل ألانين |
| Leu | لوسين |
| Ile | ايزوليوسين |
| Met | ميثيونين |
| Val | فالين |
| Ser | سيرين |
| Pro | برولين |
| Thr | ثريونين |
| Ala | ألانين |
| Tyr | تيروسين |
| ... | ... |

- يمكن أن يكون هناك 64 نوعًا من الكودونات على سلسلة الحمض النووي.
- كودونات البدء هي AUG.
- كودونات الإيقاف هي UAA و UAG و UGA.
- الحموض الأمينية التي تشفر بكودون واحد فقط هي ميثيونين وتربتوفان.
- الكودونات التي تشفر نفس الحمض الأميني تختلف في القاعدة الثالثة فقط (وهي القاعدة المهتزة).
